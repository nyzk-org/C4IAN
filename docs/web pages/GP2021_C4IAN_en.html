Project Summary C4IAN (Command, Control, Communications, Computing & Intelligence using Ad-hoc Networks) is a scalable communications solution for tactical teams. It builds a mobile ad-hoc network on-the-go between command centers and units allowing them to communicate effectively and securely while moving freely, without the need for any infrastructure. Our system runs on handheld tactical devices, robots, drones, or any other Linux-based platform. It provides both a platform-independent network layer which routes data through dynamically changing and scalable topologies, and an application for tactical teamsâ€™ units and command centers.
